레이아웃의 위치
위치 설정방법
static (정적 위치) - 정상적인 흐름에 따른 배치
	default 값이라서 쓰나 안 쓰나 똑같다.
relative (상대적 위치) - 정상적인 위치가 기준점이 된다.
	원래 있어야할 위치를 기준으로 움직임
absolute (절대적 위치) - 컨테이너의 원점이 기준점이 된다.
	'위치 지정'이 된 가까운 상위 요소를 기준으로 입력된 위치에 해당 요소가 배치
	(기준 요소가 없으면 전체 화면을 기준으로 배치)
fixed (고정 위치) - 윈도우의 원점이 기준점이 된다. (윈도우의 크기나 위치 변경이 있어도 그대로)

겹치는 요소 발생 시

	요소가 배치되면서 다른 요소와 겹치는 경우, z-index 속성으로 요소의 스택 순서를 지정

	양수 또는 음수 가능
	
	스택 순서가 큰 요소가 낮은 요소 앞에 있음

	z-index 지정되지않고 겹치면, html코드에서 마지막에 배치된 요소가 맨 위에 표시

레이아웃의 overflow

overflow 속성: 자식 요소가 부모 요소의 범위를 벗어났을 때, 어떻게 처리할 것인지를 지정

visible - default 값
hidden - 부모 영역을 벗어나는 부분을 보이지 않게함
scroll - 무조건 스크롤을 만듬
auto - 내용이 넘칠 때만 스크롤을 만듬

float과 clear
하나의 콘텐츠 주위로 다른 콘텐츠들의 물처럼 흘러가는 스타일 지정

float의 용도 - 레이아웃에 많이 사용된다.
